noPoll News

  • 2015-07-08

    New stable release correcting some bugs and providing official support for Android platform with ready to use compiled libraries for all combinations of android platform (API version) and arch (cpu style). See http://www.aspl.es/nopoll/html/nopoll_android_usage.html

    See all details at the release note: noPoll 0.3.1: "A filter inside a filter" release.

  • 2015-05-07

    Added additional corrections. Updated API to get notification when a connection is ready.

    Updated TLS handling to report EWOULD errno when finished TLS handshake to better adapt application transports that expect that error code when NULL is returned when asking for a message.

    Added support to set and get Cookie header set during HTTP handshake.

    See all details at the release note: noPoll 0.2.9: "Jumping Cookie" release.

  • 2014-12-22

    General API update to allow configuring TLS/SSL process by installing handlers that allows to create the SSL context or to implement a verification postfix. Added ready to use support to implement certificate peer verification (from client and server side).

    See all details at the release note: noPoll 0.2.8: "Old Enough" release.

  • 2014-11-11

    New stable release that includes various corrections. Added new API to allow configuring TLS protocol.

    See all details at the release note: noPoll 0.2.7: "Panic station" release.

  • 2014-01-17

    New stable release that includes several fixings.

    See all details at the release note: noPoll 0.2.6: "Jumping Wave" release.

  • 2013-09-16

    New stable release that includes several fixings and new API functions.

    See all details at the release note: noPoll 0.2.5: "God save the fence" release.

  • 2013-06-06

    New stable release that includes some fixings while receiving partial frames (at the header and mask level).

    See all details at the release note: noPoll 0.2.4: "Starwaves" release.

  • 2013-04-26

    First official noPoll release ready to use in production environment.

    See all details at the release note: noPoll 0.2.3: "The Passanger" release.
