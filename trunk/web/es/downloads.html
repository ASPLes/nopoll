Paquetes debian más recientes

Utilice el siguiente repositorio APT para instalar noPoll y sus dependencias:


  # repositorio debian publico de aspl (SQUEEZE)
  deb http://www.aspl.es/debian/public squeeze/

  # repositorio debian publico de aspl (LENNY)
  deb http://www.aspl.es/debian/public lenny/

Luego ejecute apt-cache search nopoll para buscar los paquetes para noPoll.
