Página oficial de noPoll

¡Hola! noPoll es una implementación OpenSource del protocolo WebSocket (RFC 6455), escrita en ansi C, y que permite construir soluciones WebSocket puras o proporcionar soporte de WebSocket a aplicaciones TCP existentes que requieren añadir dicho soporte.

noPoll proporciona soporte para WebSocket (ws://) y para WebSocket TLS (wss://), permitiendo programación basada en mensajes (notificación en manejador) o orientada a stream de datos.

noPoll ha sido escrita para tener una librería limpia, y fácil de utilizar e integrar. Está liberada bajo las condiciones de la LGPL 2.1 (de manera que está permitido construir aplicaciones OpenSource o comerciales) y actualmente está siendo utilizada por Vortex Library, Turbulence y Core-Admin para proporcionar soporte de WebSocket a esos proyectos (BEEP sobre WebSocket).

Acerca de Aspl

Aspl (Advanced Software Production Line) es una empresa española que proporciona soporte y servicios para introducir productos open source dentro de las organizaciones, especialmente GNU/Linux.

Fundada en el 2000, su principal línea de desarrollo es proporcionar soporte para el desarrollo de software así como soporte para instalar servidores GNU/Linux que proporcionan distintos servicios.
